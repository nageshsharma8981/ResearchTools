Getting started

How it works

1Open any tool — it's a plain HTML file, nothing is installed or uploaded.
2Add an OpenAI-compatible API key once in API settings — or point it at Ollama / LM Studio for free, fully offline use.
3Paste your material, run, watch the analysis stream in, and download it as Markdown.
Privacy

Private by design

No analytics, no telemetry, no backend. Your API key and drafts live only in this browser's local storage. The only network traffic is to the LLM endpoint you configure — and, for the literature finder, the free OpenAlex scholarly index.

A quietly powerful suite for serious research. Everything runs in your browser; your keys and drafts never leave it.
