DOI Finder & Lookup

Two directions, both keyless (1 credit per lookup): paste a DOI to pull up the full paper record, or paste messy references to discover their DOIs via Crossref.

Look up a DOI

Any format works: 10.1038/nature14539, https://doi.org/10.1038/…, or doi:10.1038/…. Data comes from Crossref, with the abstract filled in from OpenAlex when Crossref lacks one.

Find DOIs for references

One reference per line, any format — even partial or messy. Each line is matched against Crossref's 160M+ records. Results show a confidence check so you can trust (or reject) each match.

Looked up one at a time to stay polite to the free API
